# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

リモートリポジトリは、複数人で共有するリポジトリ
ローカルリポジトリは、自分用にリモートリポジトリを自分のPCに取り込んだリポジトリ


## プッシュとマージの違いは何でしょうか？

プッシュは、ローカルリポジトリの編集内容をリモートリポジトリにアップすること
マージは、切ったブランチの編集内容を幹ブランチに取り込むこと


## コミットとプッシュの違い

コミットは、編集履歴を残すこと
プッシュは、コミットした編集内容をリモートリポジトリにアップすること


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

編集内容を端的かつ分かりやすく書く


## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

前者は、ローカルでの編集内容の取り込みのため、ローカルの幹ブランチをリモートの幹ブランチにプッシュする必要がある。
後者は、リモートの幹ブランチにマージしてもらえるため、次の作業時にリモートの幹ブランチをプルして、ローカルに取り込んでから作業を始める必要がある。

## コンフリクトを起こしてしまった場合、どう対処すべきですか？

編集箇所が被ってしまった人と、編集内容を相談し、取り込む内容を決める。
その後に、エディタを使って変更を上書きする。
